## 프로젝트 구조

```bash
├── config # eject
├── public # index.html, Image Files
├── script # script
└── src
   ├── components # Components View
   ├── hooks # Custom Hook
   ├── img # Image Files
   ├── pages # Page View
   ├── redux # Redux File
   └── style # CSS File
```
